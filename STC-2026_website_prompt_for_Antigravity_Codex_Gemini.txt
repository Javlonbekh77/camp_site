STC-2026 (Summer Tech Camp 2026) — Full Website Prompt for Antigravity / Codex / Gemini

ROLE:
You are an expert full-stack product engineer, UX designer, conversion-focused landing page strategist, and startup-grade frontend developer. Build a modern, attractive, production-ready website for a local summer technology education project named:

STC-2026 — Summer Tech Camp 2026

The site must help visitors understand the project, discover the camp direction that fits them, and register for one or more camp programs. It must also include a private admin panel where the organizer can see registrations, analyze demand by camp direction, export data, and manage incoming applicants.

The site language must be mainly Uzbek Latin. Keep some technical names in English where natural: Python, SQL, Power BI, Startup, Robotics, Data Analytics, Competitive Programming, Vibe Coding, etc.

PROJECT CONTEXT:
STC-2026 is a summer tech camp in Chiroqchi for motivated students, teachers, kids, and young people who want to spend their summer usefully by learning modern technology-related skills. This is not only a course. It should feel like a local youth learning movement: a place to find direction, learn real skills, ask questions, join a motivated environment, and start building a future path.

GENERAL WEBSITE GOAL:
Create a modern, high-converting, attractive landing + registration + admin platform.

The website must:
- explain STC-2026 clearly;
- show the problem: summer can pass without useful learning, many young people do not know where to start, opportunities feel limited outside big cities;
- present STC-2026 as the solution;
- show all camp directions with clear cards;
- allow users to learn more about each camp;
- include an interactive AI-style questionnaire that recommends the most suitable camp direction;
- allow users to register for one or more camps;
- allow users to paste or save the result of their “AI suitability test” inside the registration form;
- include an admin panel to see all registrations, filter by camp, view analytics, and export CSV.

TECH STACK:
Build the project using:
- Next.js with App Router
- TypeScript
- Tailwind CSS
- shadcn/ui components if available
- Framer Motion for smooth modern animations
- React Hook Form + Zod for form validation
- Supabase for database storage
- Optional: Zustand or React Context for small state management
- Optional: Recharts for admin analytics charts
- Optional: Lucide React icons

If Supabase credentials are not provided, implement a clean fallback using localStorage for development/demo mode, but the main production-ready path must be Supabase. All Supabase integration should be cleanly abstracted in a service layer so it can be replaced later.
